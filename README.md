# 🤖 AI Model Runner - تطبيق تشغيل النماذج الذكية

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Python](https://img.shields.io/badge/python-3.8%2B-blue)

تطبيق احترافي وحديث لتشغيل نماذج ONNX ودعم ملفات DLL مع واجهة رسومية جميلة وسهلة الاستخدام.

## ✨ المميزات الرئيسية

✅ **دعم كامل لنماذج ONNX** - شغل أي نموذج ONNX بسهولة  
✅ **دعم ملفات DLL** - تكامل مع ملفات DLL الخارجية  
✅ **واجهة دردشة احترافية** - اتصل مع النموذج بطريقة طبيعية  
✅ **معلومات تفصيلية** - اعرض معلومات كاملة عن المدخلات والمخرجات  
✅ **إدارة النماذج** - حمل وأدر نماذجك بسهولة  
✅ **معالجة متعددة الخيوط** - استدلال سريع دون تجميد الواجهة  
✅ **تصدير المحادثات** - احفظ محادثاتك بصيغة JSON أو نص  
✅ **إعدادات قابلة للتخصيص** - خصص التطبيق حسب احتياجاتك  
✅ **واجهة رسومية داكنة احترافية** - تصميم حديث وسهل الاستخدام  

## 🚀 البدء السريع

### المتطلبات

- Python 3.8 أو أحدث
- Windows / Linux / macOS

### التثبيت

1. **استنساخ المستودع**
   ```bash
   git clone https://github.com/ahmedlofee2030/ai-model-runner.git
   cd ai-model-runner
   ```

2. **إنشاء بيئة افتراضية**
   ```bash
   python -m venv venv
   
   # في Windows
   venv\Scripts\activate
   
   # في Linux/macOS
   source venv/bin/activate
   ```

3. **تثبيت المتطلبات**
   ```bash
   pip install -r requirements.txt
   ```

4. **تشغيل التطبيق**
   ```bash
   python main.py
   ```

## 📖 دليل الاستخدام

### 1️⃣ تحميل نموذج ONNX

1. اذهب إلى تبويب "📦 إدارة النماذج"
2. اضغط على زر "📁 تحميل نموذج ONNX"
3. اختر ملف ONNX من جهازك
4. سيظهر النموذج في القائمة تلقائياً

### 2️⃣ تحميل ملفات DLL

1. في نفس التبويب، اضغط على "تحميل ملف DLL"
2. اختر ملف DLL
3. سيتم تحميله وحفظه في مجلد `dlls/`

### 3️⃣ الدردشة مع النموذج

1. انتقل إلى تبويب "💬 الدردشة"
2. اكتب رسالتك في صندوق الإدخال
3. اضغط "إرسال" أو اضغط Enter
4. سيقوم النموذج بمعالجة الرسالة وإرسال النتيجة

### 4️⃣ عرض معلومات النموذج

1. انتقل إلى تبويب "📊 معلومات النموذج"
2. سترى جداول تفصيلية عن:
   - مدخلات النموذج (الاسم، الشكل، النوع)
   - مخرجات النموذج
   - إحصائيات عامة

### 5️⃣ الإعدادات

1. اذهب إلى تبويب "⚙️ الإعدادات"
2. اختر من الخيارات المتاحة:
   - مزود الحساب (CPU/CUDA/TensorRT)
   - حد أقصى للمحادثة
   - حجم الخط
   - الوضع الداكن
   - عدد المعالجات
   - تفعيل التحسين
3. اضغط "حفظ الإعدادات"

## 📝 الإعدادات (config.json)

يتم إنشاء ملف الإعدادات تلقائياً عند أول تشغيل:

```json
{
  "provider": "CPU",
  "max_history": 20,
  "font_size": 11,
  "dark_mode": true,
  "num_threads": 4,
  "optimization": true
}
```

### شرح الإعدادات:

- **provider**: مزود الحساب (CPU/CUDA/TensorRT)
- **max_history**: الحد الأقصى لعدد رسائل المحادثة المحفوظة
- **font_size**: حجم الخط في البكسل
- **dark_mode**: تفعيل الوضع الداكن
- **num_threads**: عدد الخيوط المستخدمة للمعالجة
- **optimization**: تفعيل تحسين الأداء

## 🐛 معالجة الأخطاء الشائعة

### الخطأ: "No module named 'PyQt6'"
```bash
pip install --upgrade PyQt6
```

### الخطأ: "Failed to load ONNX model"
- تأكد من أن ملف ONNX صحيح
- حاول تحميل نموذج مختلف
- تحقق من سجلات الأخطاء

### الخطأ: "CUDA not available"
- غير مزود الحساب إلى CPU من الإعدادات
- أو قم بتثبيت CUDA الموافق
